
# MyCinema (Delphi VCL)

Минималан VCL пројекат са једном формом (све контроле се праве у коду).
Подржане су све функције које сте тражили: пријава, избор филма, опције (град/датум/време/тип),
избор седишта 8x8, резиме + чекирање начина плаћања, и преглед/измена/отказ резервација.

## Како да покренете
1) Отворите Delphi (RAD Studio).
2) File → Open... и изаберите `MyCinema.dpr`.
3) Притисните Run (F9).

**Лозинка:** username: `admin`, password: `1234`.

Све контроле се креирају у `uMain.pas` → `BuildUI`. Нема потребе да ручно додавате компоненте.
